Состоит из цельносварной рамы, на которой размещены элементы для установки элементов шасси, опорных, поддерживающих катков, механизм гидравлического натяжения гусеничной ленты, кронштейны для установки дизельного двигателя, блока радиаторов, трансмиссии, опоры для кабины, облицовки. Ходовая трактора состоит из двенадцати индивидуально подрессоренных с помощью торсиона опорных кареток, по шесть на каждую сторону. Так же на каждой стороне установлены по три поддерживающих катка для уменьшения прогиба гусеничной ленты и снятие критических углов провисания. Для упрощения обслуживания гусеничная лента натягивается с помощью гидравлического цилиндра до нужного эксплуатационного размера. Гусеничная лента состоит из стальных траков и резиноармированных пальцев (наплавлены полиуретаном для увеличения ресурса гусеничной ленты до 5 000 моточасов), для уменьшения трения металлических частей трака и пальца, устранения из места контакта земли и песка (абразива).
